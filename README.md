# MACHINE LORD RAP STATION ⚡️🏭

A professional-grade procedural hip-hop workstation built with C++ and the JUCE framework. This application generates full song arrangements, synthesizes instruments from pure DSP math, and features a modular "Vox Lab" for per-section vocal recording and procedural chopping.

## Key Features 🚀

- **Pure DSP Synthesis**: 100% mathematical synthesis for Kicks (808-style), Snares, Claps, metallic Hi-Hats, Sub-Bass, and Polyphonic Pads. No external samples required.
- **Procedural Arrangement Engine**: Automatically generates matching motifs across song sections (Intro, Verse, Hook, Outro).
- **Modular Section Re-Gen**: Independently regenerate the pattern of any specific section in the timeline without affecting the rest of the song.
- **The Vox Lab**: Record unique vocal takes for every section of the song. The engine algorithmically chops and screws your recordings in sync with the beat.
- **Harmonic Intelligence**: All generated elements (melodies, chords, and vocal triggers) are scale-aware and matched to the chosen Key and Scale.
- **Professional Workflow**: Real-time BPM control (40-180), adjustable Swing, integrated 12-bit Bitcrusher for Lo-Fi grit, and high-quality WAV export.

## Tech Stack 🛠

- **Language**: C++17
- **Framework**: JUCE (Audio, GUI, DSP modules)
- **Architecture**: Modular Procedural Generation (Rhythmic DNA & Patch DNA)
